# Ejecutar en desarrollo.
---------------------------------
1. Clonar el repositorio
2. Ejecutar comando
```
npm i
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. tener insataldo Docker
5. Levantar la vase de datos
```
docker-compose up -d
```
6. Reconstruir la base de datos
```
http://localhost:3000/api/v2/seed
```
## Stack usado
* MongoDB
* Nest
